WebXR:

👓 create VR and AR experiences

🤘 with open web technologies

🚀 and deliver them through the browser

How?

WebGL is an existing browser API for rendering 3D graphics in a <canvas>.

WebXR is a new API for interacting with 3D scenes from VR/AR-enabled devices.

Why?

"It's just a web page…"

🌏 Viewable on any device with a browser

🛠 Reuses existing web tools and skills

🔗 Easy to integrate with existing web content

Why not?

👨‍🔬 Experimental technology

😢 Performance is worse than native

👾 Actually… it's a bit more than a web page
